# 🚀 Projeto POO em Java - Sistema de Biblioteca e Banco

## 📚 Descrição
Projeto desenvolvido para a disciplina de **Linguagem de Programação Java** - Engenharia Informática - UNIKIVI

- `banco/` - Sistema bancário com herança
